# LaraStuck
Stack Overflow versi Indo

## Installation
LaraStuck butuh [Node.js](https://nodejs.org/).

Cara :
```sh
$ cp .env.example .env
```

Isi .env sesuai environment yang digunakan.

```sh
$ php artisan key:generate
$ composer install
$ npm install && npm run dev
$ php artisan migrate --seed
```
